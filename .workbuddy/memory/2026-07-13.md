# 2026-07-13 工作日志

## 修复：菜品列表真机显示为单列的问题
- **文件**: `pages/food/index.scss`
- **原因**: `.food-list` 使用了 `gap: 24rpx` + `.food-item` 使用 `width: calc(50% - 12rpx)`。`gap` 在微信小程序真机 Flexbox 中兼容性差，加上 `calc(50% - 12rpx)` 的亚像素舍入问题，导致两个元素总宽度恰好或略超 100%，第二个元素被挤到下一行。
- **修复方案**: 移除 `gap`，改用 `justify-content: space-between` + `width: 48%` + `margin-bottom: 24rpx`，兼容性更好。
- **教训**: 微信小程序真机调试时，Flexbox 的 `gap` 属性不可靠，应优先使用 `justify-content: space-between` 或 margin 方案。
